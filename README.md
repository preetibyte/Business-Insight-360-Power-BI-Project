# Overview  

The Business 360 Dashboard is an end-to-end data analytics solution built using Power BI to provide a comprehensive view of business performance across multiple departments such as Finance, Sales, Marketing, and Supply Chain.
This project transforms raw data into actionable insights, enabling stakeholders to make informed decisions.  

## Objectives  
- Provide a 360-degree view of business operations.
- Track KPIs and performance metrics across departments.
- Enable data-driven decision-making.
- Improve visibility into revenue, costs, and profitability.

## Tools & Technologies  
Power BI – Data visualization & dashboard creation  
SQL – Data extraction and transformation  
Excel/CSV – Data sources  
DAX (Data Analysis Expressions) – Measures & calculations  
Power Query – Data cleaning and shaping  

## Dashboard Features

### Executive Overview  
- High-level KPIs (Revenue, Profit, Growth %)
- Year-over-Year comparison
- Trend analysis

### Finance View  
- Profit & Loss statement
- Cost breakdown
- Margin analysis

### Sales View  
- Sales performance by region/product
- Top & bottom customers
- Sales trends

### Marketing View  
- Campaign performance
- Customer segmentation
- Conversion rates

### Supply Chain View  
- Inventory levels
- Order fulfillment
- Delivery performance
